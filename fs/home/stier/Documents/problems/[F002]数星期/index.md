# Description

Florance想知道从2023-06-01开始算起，到某一个日期，自己可以过几个完整的星期

# Format

## Input

输入一个日期，格式为 `yyyy-mm-dd`

## Output

输出过了的完整的星期数

# Samples

```input1
2023-07-01
```

```output1
3
```

# Tips

是完整的一星期，不是完整的7天
